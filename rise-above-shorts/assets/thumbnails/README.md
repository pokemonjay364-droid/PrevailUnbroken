# Thumbnails for Rise Above Shorts Video Series

This folder contains the thumbnail images for each video in the Overcoming Shorts Series.

## Files

- `01-3-second-rule.jpg` - The 3 Second Rule
- `02-worst-day-teacher.jpg` - Your Worst Day is Your Best Teacher
- `03-fall-7-rise-8.jpg` - Fall 7 Times, Rise 8
- `04-bamboo-tree-secret.jpg` - The Bamboo Tree Secret
- `05-you-are-stronger.jpg` - You Are Stronger Than You Know

## Specifications

- **Format:** JPEG (.jpg)
- **Recommended Size:** 1280 x 720 pixels (16:9 aspect ratio)
- **File Size:** Optimized for web (50-200KB per image)
- **Quality:** High-resolution for clarity on all devices

## Adding Your Thumbnails

1. Save your thumbnail images with the exact filenames listed above
2. Place them in this folder
3. Commit and push to your repository
4. The landing page will automatically display them
