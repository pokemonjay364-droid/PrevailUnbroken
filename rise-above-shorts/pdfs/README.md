# PDFs - Companion Guides and Resources

This folder contains downloadable PDF resources for the Rise Above Shorts series.

## Files

- `Overcoming-Shorts-Series-Companion-Guide.pdf` - The complete companion guide with exercises, reflections, and action steps for each video

## PDF Specifications

- **Format:** PDF
- **Recommended Size:** Under 50MB for optimal loading
- **Content:** Should include:
  - Introduction to the series
  - Individual lesson guides for each of the 5 videos
  - Reflection questions and exercises
  - Action steps and implementation guide

## Adding Your PDF

1. Place your PDF file in this folder
2. Ensure the exact filename: `Overcoming-Shorts-Series-Companion-Guide.pdf`
3. Commit and push to your repository
4. The landing page will automatically include the download link
5. Users can download it from the "Download Your Companion Guide" section

## Download Link

The PDF will be accessible at:
```
https://your-site.github.io/rise-above-shorts/pdfs/Overcoming-Shorts-Series-Companion-Guide.pdf
```
